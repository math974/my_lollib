int my_issapce ( char  c)

Check if we have to do with a space character.

This function allows you to test whether a character is considered a space character or not. The following characters are considered as space characters: '\ t', '\ n', '\ v', '\ f', '\ r' and space (''). Compared to the isblank function, the isspace function therefore also considers the characters '\ n', '\ v', '\ f' and '\ r'.
