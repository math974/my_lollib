int my_isprint ( char  c)

Checks whether a character is printable or not.

This function is used to test whether a character is displayed (is printable) or not. Among these characters we find all the characters having a graphic representation (those for which the isgraph function returns true) as well as the space character (code 0x20). In other words, it is all the ASCII characters, except the control characters (which can be checked via the iscntrl function).

Parameters
cthe character to be tested
Returns
The return value must be interpreted as a Boolean value.
