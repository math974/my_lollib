int my_ispunct ( char  c)

Check if we are dealing with a punctuation character.

This function allows you to test whether a character is considered an ASCII punctuation character. The following characters are considered as ASCII punctuation characters:! "# $% & '() * +, -. / :; <=>? @ [] ^ _` {|} ~. This set of characters corresponds to characters having a graphic representation (testable via the isgraph function) but not being alpha-numeric (testable via the isalnum function).

Parameters
cthe character to be tested
Returns
The return value must be interpreted as a Boolean value.
