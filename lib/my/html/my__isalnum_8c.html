int my_isalnum ( char  c)

Checks if a character is alpha-numeric.

This function allows you to test whether a character is alpha-numeric or not. A character will be considered alpha-numeric if it is a letter (lowercase or uppercase) or a number. Reminder: the functions of this library only process pure ASCII characters. Accented letters will therefore not be considered as alpha-numeric characters.
